◆ servo_stop()

void servo_stop ( )

Functions for stop work servo motor.

When the servo_stop() function is started, all controls and communication with the servo motor are blocked. To restore communication, you need to call the servo_start() function. Use this function when you need to shut down the all motors quickly.
